/* ARCHITECTURE */

Pattern: Modular JavaScript with clear separation of concerns
Modules:
- index.js: Entry point and initialization
- lib/zotero.js: Zotero API access
- lib/exporter.js: Export orchestration
- lib/format.js: Metadata generation
- lib/ui.js: User interface integration
- lib/utils.js: Logging and utilities

Data Flow:
User → UI → Exporter → Format → BagIt ZIP → Voyant Tools
